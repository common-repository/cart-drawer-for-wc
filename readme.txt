== Description ==

The WooCommerce order process can be cumbersome and time-consuming. However, today you can optimize every step of the customer experience! Cart Drawer for Woocommerce helps view items in the cart, manage the cart, and speed up the checkout process for customers.

Users may access cart items from anywhere on your site using the Cart Drawer for Woocommerce plugin. Your Woocommerce website will automatically show a side cart popup with animation whenever users add an item to the cart. With an interactive design, they can manage their cart with only a few clicks.

= Features of Cart Drawer for Woocommerce Plugin =
 
* Quick Setup
* View the Side Cart from any page on your website.
* Remove newly added items without having to reload the page.
* Show the total price of products in the Cart
* Counter number of already added product
* Subtotal of all products on the bottom
* Show product price and quantity on the side cart
* Apply Coupon code 
* Mobile Friendly


== Installation ==
1. Go to WordPress admin then go to Plugins => Add new => Upload
2. Then select the installable zip file.
3. Upload and active the plugin. After that you can go to setting page to adjust visualization and functional part.

Method 2 (Upload via FTP):
1. Upload the plugin folder to your server plugin directory. Do not upload the zip file. It should be directly the folder.
2. After this, go to WordPress admin and go to Plugins => Installed Plugins
3. If the plugin is successfully uploaded then you will find "Cart Drawer for WooCommerce" here.
4. Active the plugin. After that you can go to setting page to adjust visualization and functional part.
